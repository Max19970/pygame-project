"FeelABit"


Автор: Шуберт Максим Андреевич

Описание идеи:
"FeelABit" - это сайт-форум, где пользователи смогут публиковать свою музыку и выражать
свои мысли по поводу неё с помощью комментариев. Также на главной странице будет выведены
топ 5 самых популярных треков за всё время. Для пользования сайтом будет
необходима регистрация и/или авторизация.

Техническое задание. 
Цель: создать сайт-форум для публикации аудиофайлов с возможностью авторизации на сайте и
комментирования от имени авторизованного пользователя

Подробное описание приложения:
  ----------------------------------------------------------------------------------------------
    • Структура проекта
      1) main.py - файл, в котором описана вся логика всех страниц сайта
      2) Папка templates:
          • base.html - базовый шаблон, который наследуют все остальные страницы сайта
          • audio_x.html - шаблон страницы для публикации аудиофайлов и редактирования 
             информации о них.
          • title.html - шаблон главной страницы сайта
          • login.html - шаблон страницы авторизации
          • register.html - шаблон страницы регистрации
          • main.html - шаблон страницы с публикациями
      3) Папка static:
          a) Папка css:
              • style.css - стилевой файл для базового шаблона
      4) Папка forms:
          • user_forms.py - файл с программным описанием форм, относящихся к пользователю
            (авторизации и регистрации)
          • audio_forms.py - файл с программным описанием форм, относящихся к публикуемым
            файлам
      5) Папка data:
          • __all_models.py - сборник моделей для работы с БД
          • db_session.py - файл, используемый для подключения к БД и работы с ней
          • audio.py - файл, описывающий таблицу с информацией об аудиофайлах в БД
          • users.py - файл, описывающий таблицу с информацией о пользователях в БД
      6) Папка db:
          • data.db - БД с информацией о пользователях и опубликованных аудиофайлах
  ----------------------------------------------------------------------------------------------
    • Сам проект состоит из титульной страницы, на которую пользователь попадает при открытии
      сайта, и главной страницы, на которой авторизованные пользователи могут публиковать
      ауйдиофайлы или комментировать уже опубликованные файлы.
    • На всех страницах сверху расположена навигационная часть, с помощью которой можно попасть
      на титульную и главную страница сайта, а также кнопки для авторизации или регистрации
      пользователя
    • На титульной странице расположен логотип и список из 5 самых популярных аудиофайлов на
      сайте.
    • Чтобы полноценно работать с сайтом, пользователь должен зарегистрироваться и
      авторизовываться при последующих посещениях.
    • Как только пользователь будет авторизован, текущая страница будет сменена на главную, в
      которой и заключается весь основной функционал
    • Главная страница представляет собой ленту опубликованных на сайт аудиофайлов. При каждой
      публикации располагаются ссылка для просмотра комметариев, оставленных под записью, и
      поле для написания комментария. А для пользователей, оставивших запись в ленте, также
      доступна кнопка для удаления публикации.
      Также на самом верху главной страницы находится кнопка для добавления записи в ленту.
      Только авторизованные пользователи могут добавлять записи.
  ----------------------------------------------------------------------------------------------
